Danae Valenza Interview

"My work for NEW is Colour Piano for Chromatic Portraits, so it's basically a baby grand piano that's been modified and I've worked with a team of people, technicians, piano technicians, electricians, musicians, to make this sculpture that can be activated at any time."

"A lot of the work that I do involves making music, or the act of creating music, visual in some way. Wanting to capture a song in a portrait I think that was where it started from. And then realising that to capture a song in a moment, in a photographic portrait, I would need to create an instrument that would enable to do that."

"Each key on the piano has been rigged up to a different coloured light that's been hand painted and from there we're using the instrument to make chromatic portraits. So using long exposure analogue photography we can then take portraits of songs"


Web Design And Interactivity Assignment

Danae Valenza’s work that was seen in ACCA’s NEW14 Exhibition in 2014 is an interactive technological modification of an anologue musical instrument which creates an audio to musical visualisation. Although Valenza uses this modified instrument as a necessary tool to create long exposure photographs of the lights each song creates, I see the ‘Colour Piano’ as a powerful and inspiring artwork by itself.

As part of my second assignment for ACG206 - Web Design and Interactivity, I have chosen this installation artwork for me to reinterpret into a digital form of a website. The website will use two webpages (Home and About) and I will use the knowledge of html I have learnt this semester to build these pages. People often use the internet to listen to songs and with this website, musicians and contemporary piano enthusiasts can enjoy using an interactive 'digital piano' based on Valenza's work while listening to classic solo jazz piano pieces.
